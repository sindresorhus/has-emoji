# has-emoji

> Check whether a string has any emoji

This can be useful when you need to prevent emoji in user input, for example, for usernames.


## Install

```
$ npm install has-emoji
```


## Usage

```js
const hasEmoji = require('has-emoji');

hasEmoji('Unicorn ðŸ¦„');
//=> true

hasEmoji('Cat');
//=> false
```
